# Varsco Pressure Washing — Website

Live site: **https://varscoservices.com** (hosted on Netlify)

- `index.html` — the entire website. Single self-contained page; all images are
  embedded inline as base64. The only external dependency is Google Fonts.
- Recovered from the live Netlify deploy on **2026-05-31** (the site was originally
  published by drag-and-drop, so the deployed file *is* the source).

## Editing & publishing
Edit `index.html` here, commit the change with git, and (once GitHub + Netlify
continuous deploy is connected) the live site updates automatically on push.

Open `index.html` for the `SWAP-INS` checklist of placeholders to replace before
launch (phone number, photos, reviews, etc.).
